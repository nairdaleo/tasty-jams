How Chlorophyll supports iOS accessibility features

Chlorophyll is built to work well for everyone. The features below describe how the app supports the accessibility settings built into iOS and iPadOS. If something doesn't work for you, please get in touch — every report helps.

Supported

👁

VoiceOver

All icon-only buttons have descriptive labels (Settings, Add plant, More options, etc.). Interactive task rows announce their full state: plant name, task name, due date, and whether they've been logged today. Quick-action buttons say "Water for Monstera — done today" and hint "Double tap to undo" so VoiceOver users know exactly what will happen.

📸 Screenshot — VoiceOver label on quick-action button
🎙

Voice Control

Voice Control identifies tap targets using the same accessibility labels as VoiceOver. Every button and action in the app has a unique, descriptive label, so you can say "Tap Add plant", "Tap Settings", or "Tap Water" and it will work.

📸 Screenshot — Voice Control name grid on plant list
Aa

Larger Text

All text in Chlorophyll uses iOS Dynamic Type text styles (Body, Headline, Caption, etc.) and scales automatically with the text size you've set in iOS Settings → Accessibility → Display & Text Size. At large Accessibility sizes, plant names and collection names wrap to multiple lines instead of clipping.

📸 Side-by-side — default text size vs. Accessibility XL
🧘

Reduce Motion

When Reduce Motion is enabled in iOS Settings → Accessibility → Motion, all non-essential animations in the app are disabled: quick-action button scales, task completion transitions, search bar slide-in/out, and filter chip toggles all become instant. Haptic feedback is also lightened. Content and functionality are identical — only the motion is removed.

📸 Illustration — Reduce Motion on vs. off
🌙

Dark Interface

Chlorophyll ships with seven dark themes: Noche, Yoru, Layl, Kaamos, Tlapalteotl, Usiku, and Alebrije. Each one is a fully dark-mode experience with light text on dark surfaces. The active theme forces the entire app — including system sheets and navigation — into dark mode.

📸 Mosaic — dark theme palette showcase
◑

Sufficient Contrast

Two themes are designed specifically for maximum contrast and verified against WCAG 2.1 AA (4.5:1 minimum ratio for normal text). Every text colour, accent, and task-type colour in these themes has been mathematically confirmed to pass.

🔆 Nítido — Light · ≥6:1 🖤 Ónice — Dark · ≥9:1

Both themes are available in Settings → Appearance and are labelled "WCAG AA · Accessibility" so they're easy to find.

📸 Nítido vs. Ónice with contrast ratio callouts
◆

Differentiate Without Colour

Task types (watering, fertilizing, repotting, pruning) are always identified by both a colour and an SF Symbol icon. Colour is never the only signal — the icon alone is sufficient to tell tasks apart. This means the app is usable even when colour perception is limited.

Not Applicable

💬

Captions Chlorophyll contains no video content. There are no videos, video playback, or video-based tutorials in the app, so closed captions are not applicable.

🔊

Audio Descriptions Audio descriptions are for videos that convey information visually without narration. Chlorophyll has no video content, so audio descriptions are not applicable.

Something Not Working?

If an accessibility feature isn't working as described — or if you use a feature not listed here and Chlorophyll doesn't support it well — please reach out. Accessibility feedback is treated as a bug report, not a feature request.
